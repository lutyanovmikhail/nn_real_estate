# 🏠 Оценка стоимости квартир в Нижнем Новгороде

Ансамблевая модель для предсказания цен на вторичном рынке недвижимости Нижнего Новгорода. Данные собраны самостоятельно с трёх источников, проект включает полный цикл: парсинг → очистка → статистический анализ → feature engineering → обучение и валидация модели.

---

## Результаты

### Ансамбль LightGBM (0.6) + CatBoost (0.4)

| Метрика | Значение |
|--------|----------|
| R² | **0.895 ± 0.007** |
| MAE | **1.21 ± 0.02 млн ₽** |
| MAPE | **11.34 ± 0.64%** |
| Метод оценки | 5-fold Stratified CV |

### Сравнение моделей

| Модель | R² | MAE | MAPE | Стабильность (std R²) |
|--------|-----|-----|------|----------------------|
| CatBoost | 0.895 | 1.31 млн ₽ | 11.34% | ± 0.015 |
| **Ансамбль** | **0.895** | **1.21 млн ₽** | **11.34%** | **± 0.007** |

Ансамбль улучшил MAE на **100к ₽** и вдвое повысил стабильность предсказаний.  
Веса подобраны grid search: LightGBM 0.6 / CatBoost 0.4.

> Для контекста: медианная ошибка Zillow на рынке США ~7%. Нижний Новгород — провинциальный рынок с ~6к объектов и высокой внутрирайонной дисперсией цен, что объективно сложнее для предсказания.

## Данные

**Источники:** gipernn.ru, cian.ru, avito.ru  
**Объём:** 5 930 объявлений (после дедупликации)  
**Период:** апрель 2025  
**Сбор:** самостоятельный парсинг трёх источников с нуля

Ключевые признаки:
- Площадь (общая, кухня), количество комнат, этажность
- Геопризнаки: расстояние до центра и метро (геокодинг через Nominatim + haversine)
- Район, материал стен, год постройки
- Target encoding: средняя цена м² по сегменту района (честный, без утечки)

**Пропуски:** 52% объектов без `distance_to_metro` — обрабатываются моделями нативно.

## Статистический анализ рынка

### Распределение цен
Цены имеют выраженную правостороннюю скошенность (skewness = 4.1, kurtosis = 33.2).  
Нормальность отвергнута двумя тестами:
- **Shapiro-Wilk** (n=100): W = 0.706, p = 1.34e-22
- **Колмогоров-Смирнов** (n=500): D = 0.185, p = 1.61e-12

Логарифмирование цены статистически обосновано: log(price) приближается к нормальному распределению.

### Классификация районов

Попарное сравнение 8 районов (28 пар): bootstrap CI с поправкой Бонферрони (α = 0.05/28 = 0.0018) + Cohen's d.

| Сегмент | Районы | Cohen's d vs остальных |
|---------|--------|----------------------|
|  Elite | Нижегородский | d > 1.1 против всех |
| ️ Upper-Mid | Советский | d > 0.8 против Mass-Market |
|  Mid | Канавинский | d = 0.44–0.77 |
|  Mass-Market | Сормовский, Автозаводский, Московский, Ленинский | d < 0.12 внутри группы |

**Ключевые находки:**
- Нижегородский район дороже всех остальных на **10–14 млн ₽** (d > 1.1, p < 0.001)
- Четыре района Mass-Market статистически неразличимы после поправки Бонферрони
- Результаты подтверждены тремя методами: t-тест Уэлча, bootstrap CI, Cohen's d

### Анализ этажей

Bootstrap-тест (10 000 итераций) выявил иерархию:

```
Средние этажи > Последний этаж > Первый этаж
```

Разница первый vs последний: **1.3–3.5 млн ₽** (p < 0.0001).  
Подтверждено: t-тест Уэлча (p=3.3e-11), Mann-Whitney U (p=1.1e-21), Bootstrap.

## Модель

### Архитектура ансамбля

```
Входные данные
      │
      ├──► CatBoost  (нативные категории + target encoding)  × 0.4
      │
      └──► LightGBM  (нативные категории)                    × 0.6
                │
                └──► Взвешенное среднее предсказаний
```

**Таргет:** log(price) → обратное преобразование exp() для метрик  
**Функция потерь:** MAPE (прямая оптимизация целевой метрики)  
**Гиперпараметры:** подобраны через Optuna (30 trials каждая модель)

### Топ признаков по важности (SHAP)

| Признак | Важность | Интерпретация |
|---------|----------|---------------|
| total_area | 18.1% | Главный драйвер — "цена бетона" |
| avg_price_m2_segmented | 13.7% | Локальный ценовой ориентир сегмента |
| area_ratio_to_district | 9.9% | Детектор премиум-объектов |
| kitchen_area | 8.2% | Прокси для класса жилья (европланировка) |
| district | 8.1% | Фиксированный "лифт" к цене |

### Качество по ценовым сегментам

| Сегмент | MAPE | MAE |
|---------|------|-----|
| Дешёвые (< 7 млн) | 12.1% | 584к ₽ |
| Средние (7–20 млн) | 10.4% | 1.27 млн ₽ |
| Дорогие (> 20 млн) | 16.3% | 4.67 млн ₽ |

## SHAP — объяснимость модели

**Район добавляет фиксированный "лифт" к цене, но не усиливает эффект площади.**  
Dependence plot показывает: все районы дают одинаковый наклон зависимости цены от площади.

**area_ratio_to_district — детектор премиум-объектов.**  
Квартиры с ratio > 2.0 получают значительный буст — модель интерпретирует это как сигнал элитности.

**kitchen_area — прокси для класса жилья**, а не просто размер кухни.  
Большая кухня (> 12 м²) маркирует европланировку и новостройки комфорт+.

### Пример объяснения (waterfall)

Квартира: Нижегородский район, 116 м², кирпич, кухня 27 м²

| Признак | Значение | Вклад в цену |
|---------|----------|-------------|
| total_area | 116.29 м² | +0.36 |
| area_ratio_to_district | 1.584 | +0.14 |
| avg_price_m2_segmented | 272 568 ₽/м² | +0.14 |
| kitchen_area | 27 м² | +0.13 |
| district | Нижегородский | +0.08 |
| year | 16 лет | −0.10 |

Базовое предсказание: **9.97 млн ₽** → Итоговое: **26.58 млн ₽**

## Ограничения и планы

**Текущие ограничения:**
- 52% объектов без геопризнаков (геокодинг не распознал адрес)
- Avito-данные не содержат `living_area`, `year`, `material` — заполнены NaN
- Датасет ~6к объектов — для статистически значимого тюнинга нужно ~15к

**Планы:**
- Streamlit-приложение: ввод параметров → предсказание ансамблем + SHAP объяснение
- Расширение датасета до 15к через периодический парсинг
- Версия 2.0 для риелторов: фокус на масс-маркет сегмент (7–20 млн ₽)
