# 🏠 Оценка стоимости квартир в Нижнем Новгороде

Модель машинного обучения для предсказания цен на вторичном рынке недвижимости Нижнего Новгорода. Данные собраны самостоятельно с трёх источников, проект включает полный цикл: парсинг → очистка → статистический анализ → feature engineering → обучение и валидация модели.

---

## Результаты

| Метрика | Значение |
|--------|----------|
| R² | **0.895 ± 0.015** |
| MAE | **1.31 ± 0.03 млн ₽** |
| MAPE | **11.34 ± 0.51%** |
| Метод оценки | 5-fold Stratified CV |
| Переобучение (train-val gap) | 0.05 в среднем |

> Для контекста: медианная ошибка Zillow на рынке США составляет ~7%. Нижний Новгород — провинциальный рынок с ~5к объектов и высокой внутрирайонной дисперсией, что объективно сложнее для предсказания.

---

## Данные

**Источники:** gipernn.ru, cian.ru, avito.ru  
**Объём:** 5 930 объявлений (после дедупликации)  
**Период:** апрель 2025  
**Сбор:** самостоятельный парсинг трёх источников с нуля

Ключевые признаки:
- Площадь (общая, кухня), количество комнат, этажность
- Геопризнаки: расстояние до центра и метро (геокодинг через Nominatim + haversine)
- Район, материал стен, год постройки
- Target encoding: средняя цена м² по сегменту района

**Пропуски:** 52% объектов без `distance_to_metro` (адрес не распознан геокодером) — обрабатываются CatBoost нативно.

## Статистический анализ рынка

### Распределение цен
Цены имеют выраженную правостороннюю скошенность (skewness = 4.1, kurtosis = 33.2). Нормальность отвергнута двумя тестами:
- **Shapiro-Wilk** (n=100): W = 0.706, p = 1.34e-22
- **Колмогоров-Смирнов** (n=500): D = 0.185, p = 1.61e-12

Логарифмирование цены статистически обосновано: распределение log(price) приближается к нормальному, что снижает влияние выбросов и улучшает качество модели.

### Классификация районов

Попарное сравнение 8 районов (28 пар): bootstrap CI с поправкой Бонферрони (α = 0.05/28 = 0.0018, уровень доверия 99.82%) + Cohen's d.

| Сегмент | Районы | Cohen's d vs остальных |
|---------|--------|----------------------|
|  Elite | Нижегородский | d > 1.1 против всех |
| ️ Upper-Mid | Советский | d > 0.8 против Mass-Market |
|  Mid | Канавинский | d = 0.44–0.77 |
|  Mass-Market | Сормовский, Автозаводский, Московский, Ленинский | d < 0.12 внутри группы |

**Ключевые находки:**
- Нижегородский район дороже всех остальных на **10–14 млн ₽** (d > 1.1, p < 0.001)
- Четыре района Mass-Market статистически неразличимы после поправки Бонферрони — единый ценовой пул
- Все результаты подтверждены тремя методами: t-тест Уэлча, bootstrap CI, Cohen's d

### Анализ этажей

Bootstrap-тест (n=10 000 итераций) выявил неожиданную иерархию:

```
Средние этажи > Последний этаж > Первый этаж
```

Разница первый vs последний: **1.3–3.5 млн ₽** (p < 0.0001).  
Подтверждено тремя методами: t-тест Уэлча (p=3.3e-11), Mann-Whitney U (p=1.1e-21), Bootstrap (p < 0.0001).

> Вывод опровергает распространённое мнение о том, что последний этаж так же непривлекателен, как первый.

### Признак kitchen_ratio

`kitchen_ratio` (доля кухни в общей площади) показал умеренную корреляцию с ценой (r = 0.21, p = 2.4e-31). Это не эффект размера квартиры — доля нейтрализует абсолютную площадь. Высокий `kitchen_ratio ≥ 0.25` является маркером европланировки, характерной для новостроек бизнес-класса.

## Модель

**Алгоритм:** CatBoostRegressor  
**Таргет:** log(price) → обратное преобразование exp() для метрик  
**Функция потерь:** MAPE (прямая оптимизация целевой метрики)

**Гиперпараметры** (подобраны через Optuna, 50 trials):

```python
CatBoostRegressor(
    iterations=1135,
    depth=8,
    learning_rate=0.06,
    l2_leaf_reg=27,
    loss_function='MAPE',
    cat_features=['district', 'material', 'district_ready',
                  'mini_disctrict', 'material_age']
)
```

**Топ признаков по важности:**

| Признак | Важность |
|---------|----------|
| total_area | 18.1% |
| avg_price_m2_segmented | 13.7% |
| area_ratio_to_district | 9.9% |
| kitchen_area | 8.2% |
| district | 8.1% |

**Качество по ценовым сегментам:**

| Сегмент | MAPE | MAE |
|---------|------|-----|
| Дешёвые (< 7 млн) | 12.1% | 584к ₽ |
| Средние (7–20 млн) | 10.4% | 1.27 млн ₽ |
| Дорогие (> 20 млн) | 16.3% | 4.67 млн ₽ |

## Ограничения и планы

**Текущие ограничения:**
- 52% объектов без геопризнаков (геокодинг не распознал адрес)
- Авито-данные не содержат `living_area`, `year`, `material` — заполнены NaN
- Для статистически значимого подтверждения эффекта sample weights нужно ~7 500 объектов

**Планы:**
- Деплой в Streamlit: ввод параметров квартиры → предсказание цены
- Расширение датасета до 15к объектов через периодический парсинг
- SHAP-анализ для объяснения индивидуальных предсказаний
